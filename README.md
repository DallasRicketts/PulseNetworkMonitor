Pulse Network Monitor

Pulse is a lightweight, real-time network monitoring tool designed to keep your network health at a glance. Featuring a sleek system tray interface with live status updates, Pulse allows users to monitor multiple network interfaces, view connection status, and quickly identify issues before they become critical.

Features

Real-Time Network Monitoring – View current network health and traffic at a glance.

System Tray Integration – Keep Pulse running quietly in the tray without cluttering your desktop.

Visual Status Indicators – Green (healthy), yellow (attention required), red (critical error)

Pulse Overlay – Activity pulses visually indicate ongoing network scans.

Customizable Scan Settings – Set passive scan intervals, adjust alerts, and configure notifications.

External Console – Optional console window for advanced diagnostics.

Installation

Clone the repository:

git clone https://github.com/DallasRicketts/PulseNetworkMonitor.git


Open Pulse.sln in Visual Studio 2022 or later.

Build the solution and run PulseNetworkMonitor.exe.

Note: Requires .NET Framework 4.8 or later.

Usage

Launch Pulse from the system tray or directly via the executable.

Right-click the tray icon to access settings and start network scans.

Monitor network health with visual cues and optional console logs.

Adjust scan intervals and thresholds through the settings form.
